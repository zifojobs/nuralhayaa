# Prompts images IA — NAID

Ce fichier contient les prompts à utiliser pour générer (Midjourney, DALL·E, Flux) ou sélectionner (Unsplash, Pexels) les visuels du site. Les prompts sont en anglais car les modèles IA y répondent mieux, mais chaque image est accompagnée d'une note en français sur son usage.

**Charte visuelle NAID**
- Palette : vert forêt (#1C4A2D), or (#C9A961), crème (#FAF7F2), noir profond (#0E0E0E), accents terre/ocre
- Ambiance : luxe éditorial, lumière naturelle chaude, matières nobles (bois, laiton, lin, marbre), intégration subtile d'éléments afro-modernes
- Éviter : couleurs saturées, décors froids ou trop minimalistes, kitsch tropical

## Conseils de génération

1. **Midjourney v6+** : utiliser `--ar 16:9 --style raw --stylize 100` pour un rendu photographique réaliste.
2. **Flux Pro** : excellent pour les intérieurs, préférer le preset "photography realism".
3. **DALL·E 3** : plus difficile pour les intérieurs complexes, utiliser des prompts très descriptifs.
4. **Alternative gratuite** : Unsplash (tags : `luxury interior`, `boho interior`, `editorial living`) + Pexels.

**Toujours vérifier** :
- La palette correspond à la charte (vert / or / crème)
- Pas de logos ou textes hallucinés
- Visages humains à usage avec précaution (droits, cohérence) — privilégier des photos de Rokhaya quand possible
- Résolution suffisante (min 1600px sur le côté long pour desktop)
